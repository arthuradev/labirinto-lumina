## Entrega final

`v1.0.0` fecha o escopo planejado de Labirinto Lumina: um jogo web arcade original de labirinto,
coleta e perseguição, publicado no GitHub Pages e desenvolvido integralmente por IA com GPT-5.5 sob
direção humana.

## Conteúdo do jogo

- Tela inicial, tela de controles, pausa, game over e vitória.
- Três fases originais: `Circuito Aurora`, `Malha Prisma` e `Nucleo Obsidiana`.
- Movimento em grid com teclado.
- Fragmentos de luz, nós de pulso, pontuação, vidas e bônus de conclusão.
- Sentinelas originais com comportamentos simples.
- Colisão com paredes e sentinelas.
- Progressão de campanha entre fases.
- High score local via `localStorage`.
- Sons originais sintetizados via Web Audio e mute com `M`.

## Qualidade

- TypeScript, Vite, Canvas, Vitest, Playwright, ESLint e Prettier.
- `npm run check` inclui typecheck, lint, format check, testes unitários, build e smoke test e2e.
- GitHub Actions valida o projeto e publica GitHub Pages em pushes para `main`.
- Revisão final de originalidade registrada.
- Checklist final aprovado.
