# Plano de versões

Este projeto segue SemVer no formato:

```text
vX.Y.Z
```

Durante o desenvolvimento inicial, serão usadas versões `v0.x.x`.

## v0.1.0 — Fundação documental e técnica inicial

Entrega:

- documentação raiz;
- metodologia IA-first;
- política de originalidade;
- arquitetura planejada;
- setup técnico mínimo;
- tela provisória;
- CI inicial.

Gameplay: não.

## v0.2.0 — Núcleo técnico do jogo

Entrega:

- estrutura interna;
- canvas funcional;
- game loop;
- estados básicos;
- renderização inicial.

- primeira fase navegável.

## Regras

- Não criar tag se validações falharem.
- Não criar release se documentação estiver desatualizada.
- Não pular versões planejadas sem atualizar este arquivo.
- Não adicionar features fora do escopo da v1.0.0.
