# ADR 0002 — Política de originalidade e propriedade intelectual

## Status

Aceita.

## Contexto

O projeto é inspirado em mecânicas genéricas de jogos arcade de labirinto, mas será público no GitHub.

É necessário evitar risco de copyright, trademark e confusão de identidade.

## Decisão

O projeto não usará:

- nome Pac-Man;
- variações confundíveis;
- sprites protegidos;
- sons protegidos;
- mapas copiados;
- personagens parecidos;
- marcas comerciais;
- artes de jogos existentes.

A identidade será baseada em:

- luz;
- circuitos;
- energia;
- ruído;
- sentinelas;
- labirintos originais.

## Consequências positivas

- Reduz risco jurídico.
- Fortalece identidade própria.
- Melhora apresentação do projeto como criação original.

## Consequências negativas

- Exige mais cuidado criativo.
- A IA deve evitar soluções visuais óbvias demais.

## Regra para IA

Se algo parecer um clone visual ou sonoro, não usar.
