# ADR 0003 — Escopo fechado para v1.0.0

## Status

Aceita.

## Contexto

O projeto deve nascer completo dentro de um escopo realista.

O objetivo não é manter uma lista infinita de futuras features, mas concluir uma versão jogável, documentada e publicada.

## Decisão

A versão `v1.0.0` terá escopo fechado:

- 3 fases;
- jogador;
- sentinelas;
- coleta;
- pontuação;
- vidas;
- pausa;
- game over;
- vitória;
- high score local;
- áudio simples;
- GitHub Pages;
- documentação final.

Não entram:

- multiplayer;
- backend;
- ranking online;
- editor de fases;
- skins complexas;
- PWA;
- engine externa;
- mobile avançado.

## Consequências positivas

- Projeto terminável.
- Menos risco de escopo infinito.
- IA trabalha com objetivo claro.

## Consequências negativas

- Ideias interessantes ficarão fora da primeira versão.
- O projeto prioriza conclusão sobre quantidade de features.
