# Política de originalidade

## 1. Objetivo

Garantir que **Labirinto Lumina** seja um jogo original e não use materiais protegidos por copyright, trademark ou identidade comercial de jogos existentes.

## 2. Inspiração permitida

É permitido se inspirar em mecânicas genéricas:

- labirinto;
- coleta;
- perseguição;
- pontuação;
- fases;
- inimigos;
- power-up temporário;
- vitória;
- derrota.

## 3. Proibido

É proibido usar:

- nome Pac-Man;
- nomes parecidos;
- personagens parecidos;
- sprites parecidos;
- fantasmas;
- boca amarela circular;
- sons parecidos;
- mapas copiados;
- artes comerciais;
- logos;
- fontes visuais associadas;
- assets extraídos de jogos;
- qualquer material sem licença clara.

## 4. Direção criativa própria

A identidade deve seguir temas como:

- luz;
- energia;
- circuitos;
- escuridão;
- ruído;
- sentinelas;
- fragmentos;
- pulso.

Nomes internos recomendados:

```text
Jogador: Lumina
Coletável: Fragmento de Luz
Power-up: Nó de Pulso
Inimigos: Sentinelas
```

## 5. Mapas

Todos os mapas devem ser criados do zero.

Regras:

- não copiar layouts conhecidos;
- não reproduzir corredores icônicos;
- não imitar mapa clássico;
- criar geometrias próprias;
- documentar formato de fase.

## 6. Sons

Sons devem ser:

- criados para o projeto;
- gerados originalmente;
- ou CC0 com origem documentada.

Proibido criar sons que imitem efeitos sonoros de jogos comerciais.

## 7. Visual

Visual deve ser próprio.

Evitar:

- amarelo circular dominante associado a personagem clássico;
- inimigos com silhueta de fantasmas;
- estética que pareça clone;
- tela que tente reproduzir identidade de outro jogo.

## 8. Revisão antes de release

Antes de `v0.9.0` e `v1.0.0`, revisar:

- nomes;
- mapas;
- sprites;
- sons;
- README;
- screenshots;
- release notes.

Se algo parecer confundível, substituir.
